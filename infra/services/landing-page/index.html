Fácil de Integrar

Use a mesma interface da OpenAI. Troque apenas a URL e a chave.

# Substitua pela sua chave de API
curl https://api.ativadata.com/v1/chat/completions \
  -H "Authorization: Bearer SUA_CHAVE_API" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-3.5-turbo",
    "messages": [
      {"role": "user", "content": "Olá!"}
    ]
  }'

Perguntas Frequentes

Como recebo minha chave de API?

Após o pagamento, você recebe automaticamente por e-mail em segundos. Incluímos a chave, o endpoint e um exemplo de uso pronto.

É compatível com a API da OpenAI?

Sim. Basta trocar a URL base para api.ativadata.com e usar sua chave. Funciona com qualquer SDK ou biblioteca que suporte a OpenAI.

Quais modelos estão disponíveis?

Oferecemos modelos otimizados equivalentes ao gpt-3.5-turbo, com excelente performance em português. Modelos de embeddings também estão disponíveis.

Como funciona o serviço de imagens?

Envie a foto do seu produto por WhatsApp, escolha o estilo desejado e receba as imagens profissionais em até 24 horas. A primeira foto é grátis para você testar.

Posso cancelar a qualquer momento?

Sim. Todos os planos são mensais sem fidelidade. Cancele quando quiser pelo painel do Stripe.
